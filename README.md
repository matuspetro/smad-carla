# SMAD-zadanie-21
Zadanie č. 21 - Navigácia autonómneho vozidla v premávke
Cieľom zadania je vytvoriť RL agenta, ktorý sa naučí bezpečne jazdiť v simulovanej premávke s ostatnými vozidlami. Vhodné metódy na túto úlohu sú DQN a PPO. Po úspešnej implementácií jednotlivých algoritmov porovnajte prevádzkové vlastnosti jednotlivých algoritmov z pohľadu
Počet nehôd počas tréningu
Priemerný čas na dokončenie jazdy
Vplyv rôznych modelov premávky na tréning
